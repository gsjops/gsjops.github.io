# 个人网站

现代风格的纯静态个人网站，使用 HTML5 + CSS3 + JavaScript 原生开发。

## 项目结构

```
my-website/
├── index.html          ✅ 首页
├── about.html          ✅ 关于我
├── projects.html       ✅ 项目
├── blog.html           ✅ 学习笔记
├── contact.html        ✅ 联系方式
├── css/
│   └── style.css       ✅ 全局样式
├── js/
│   └── main.js         ✅ 主脚本
├── images/
│   └── hero-bg.svg     ✅ Hero 背景图
└── README.md
```

## 预览方式

直接用浏览器打开 `index.html` 即可预览，无需安装任何依赖。

## 开发进度

- [x] 第一步：项目目录结构 + 首页
- [x] 第二步：子页面基础框架（关于我 / 项目 / 学习笔记 / 联系方式）

## 技术特性

- 纯静态，无框架、无后端
- 响应式布局（手机 + 电脑）
- 深色模式切换
- 滚动入场动画
- 卡片悬停动效
- 固定导航栏 + 返回顶部
